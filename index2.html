HTML basics

Welcome to the HTMl basics page

Where you learn how to correctly code each part of a website.


Links


Links in HTML allow you to connect one webpage to another.


Create a link using the <a> tag:

<a href="https://example.com">Visit Website</a>


Images


Images in HTML are used further enhance a the description

You can display an image using the <img> tag:

<img src="image.jpg" alt="Description of image" />

Lists

Ordered List


Items appear numbered.

<ol>
            <li>First item</li>
            <li>Second item</li>
            </ol>

Unordered List


Items appear with bullet points

<ul>
           <li>Item one</li>
           <li>Item two</li>
           </ul>

Tables


Tables organize data rows and columns


<table>
             <th>Name</th>
             <th>Age</th>
             </tr>
             <tr>
             <td>John</td>
             <td>25</td>
             </tr>
             </table>

Forms

Forms allow users to enter information. They are used for logins, searches, registrations, etc.

<form action="submit.php" method="POST">
             <label>Name:</label>
             <input type="text" name="username" />
             <button type="submit">Submit</button>
             </form>

Comments


Comments let you leave notes in your HTMl code without affecting the webpage.


<!-- This is a comment -->
