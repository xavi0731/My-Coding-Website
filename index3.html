How to Use CSS


You can use CSS in three main ways:

  • Inline CSS: <p style="color:red;">Text</p>
  • Internal CSS: Using a <style> tag inside <head> (like this page)
  • External CSS: Linking a CSS file: 
    <link rel="stylesheet" href="styles.css">
